Popover Widget

Bootstrap-like popover implemented in pure JavaScript

Interactive Demo

Click the buttons below to toggle popovers. Each popover has a title and content, and is centered horizontally above the triggering element.

Widget Features

  • Pure JavaScript implementation (no jQuery)
  • Horizontally centered above trigger element
  • Customizable title and content
  • Click outside to close
  • Smooth animations
  • Fully responsive

Project Setup

  • Built with Webpack 5
  • Tested with Jest & JSDOM
  • Deployed via GitHub Actions
  • Uses Babel for ES6+ support
  • Includes ESLint for code quality

Implementation Details

How to use the Popover class:

// Create a new popover instance
const popover = new Popover(element, {
    title: 'Popover Title',
    content: 'Popover content text here'
});

// Show the popover
popover.show();

// Hide the popover
popover.hide();

// Toggle the popover
popover.toggle();

// Remove the popover completely
popover.destroy();
